Github setup(untested)
	From within Visual Studio 2013:
	Select File -> Open -> Open From Source Control
	The Team Explorer window will come up, one of the sections is "Local GIT repositories". In that section choose "Clone".
	That will bring up a path selection textbox and directory browser. Simply replace the default path with the git URL (in your case "https://github.com/Villagetronic/V-Base.Compatibility")
